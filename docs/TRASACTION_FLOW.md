# Transaction Flow

## Overview

Transactions follow a strict processing pipeline.

## Transfer Flow

1 User initiates transfer  
2 System validates user balance  
3 Fee calculation occurs  
4 Transaction recorded  
5 Ledger entries created  
6 Wallet balances updated  
7 Notification sent

## Deposit Flow

User → Mobile Money → MwalaPay → Wallet Credit

## Withdrawal Flow

User → MwalaPay → Mobile Money Provider → User Account

## Key Guarantees

Atomic transactions  
Ledger integrity  
Audit traceability
