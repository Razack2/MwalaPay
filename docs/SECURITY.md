# Security Architecture

## Overview

Security is fundamental to the MwalaPay platform.

## Authentication

Users authenticate using:

- Phone number
- Password or PIN
- Two-Factor Authentication

## Encryption

Sensitive data is protected using:

TLS encryption for network traffic  
Database encryption for stored information

## Fraud Prevention

Monitoring systems detect:

- abnormal transaction patterns
- unusual login attempts
- large withdrawals

## Access Control

Role-based permissions restrict system access.

Admin roles include:

- system admin
- financial auditor
- support staff
