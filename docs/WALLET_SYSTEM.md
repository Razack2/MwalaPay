# Wallet System

## Overview

The wallet system allows users to store and manage funds in multiple currencies.

Supported wallet types:

- MWK wallet
- USD wallet
- Cryptocurrency wallets

## Features

- Deposit funds
- Withdraw funds
- Send money
- Receive money
- Pay merchants
- Online payments

## Balance Calculation

Wallet balances are derived from ledger entries.

Balances are never manually edited.

## Supported Operations

Deposit  
Withdrawal  
Transfer  
Merchant payment  
Currency conversion

## Multi-Currency Support

Each user can hold multiple wallets:

Example

User
 ├ MWK Wallet
 ├ USD Wallet
 └ Crypto Wallet
