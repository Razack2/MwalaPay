# KYC and AML Compliance

## Overview

MwalaPay complies with financial regulations and anti-money laundering requirements.

## KYC Levels

Level 1
Basic user registration.

Level 2
Government ID verification.

Level 3
Full verification with proof of address.

## AML Monitoring

The system monitors:

- high value transactions
- suspicious transfers
- unusual patterns

## Regulatory Compliance

The platform aligns with guidelines from financial regulators such as the Reserve Bank of Malawi.
