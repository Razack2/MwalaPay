# External Integrations

## Overview

MwalaPay integrates with financial service providers to enable deposits, withdrawals, and payments.

## Mobile Money

Supported providers include:

- Airtel Money
- TNM Mpamba

## Banking Systems

Bank integrations allow users to:

- link bank accounts
- transfer funds
- withdraw funds

## International Payment Networks

MwalaPay supports integrations with global payment providers including:

- PayPal
- Mastercard

These integrations enable online purchases and cross-border payments.

## Integration Architecture

External systems connect through secure APIs and webhook notifications.
