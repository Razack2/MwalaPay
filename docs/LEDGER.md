# Financial Ledger

## Overview

The ledger is the financial backbone of MwalaPay.

It records every transaction using **double-entry accounting**.

## Why Double Entry

Double-entry accounting ensures that:

Total debits = total credits

This prevents financial inconsistencies.

## Example Transfer

User A sends MWK 10,000 to User B.

Entry 1
Debit: User A Wallet
Credit: User B Wallet

Entry 2
Debit: User A Wallet
Credit: Platform Fee Account

## Ledger Properties

Immutable  
Auditable  
Traceable  
Tamper-resistant

## Correction Mechanism

Ledger entries cannot be modified.

Corrections require **reversal transactions**.
